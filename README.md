# OIBSIP_task3

## AIM
To develop a Temperature Converter Website that converts values between Celsius and Fahrenheit.To provide accurate results with user input validation and clear output display.

## ALGORITHM
1. Start and take temperature input from the user.
2. Validate that the input is a numeric value.
3. Select the input unit (Celsius or Fahrenheit).
4. Apply the conversion formula based on the selected unit.
5. Display the converted temperature and end the program.
